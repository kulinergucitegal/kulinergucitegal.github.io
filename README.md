# RM APAYAA - Kuliner Guci Tegal

Repository ini berisi source website [kulinergucitegal.com](https://kulinergucitegal.com), website resmi RM APAYAA untuk informasi rumah makan, menu, fasilitas, reservasi WhatsApp, dan artikel pendukung wisata Guci Tegal.

Website dibangun dengan Jekyll dan theme Chirpy sebagai fondasi, lalu disesuaikan dengan layout restaurant custom.

## Stack

- Jekyll
- Ruby dan Bundler
- jekyll-theme-chirpy
- GitHub Pages
- GitHub Actions untuk build dan deploy
- Google Analytics melalui konfigurasi `_config.yml`
